460  /*!
461  * The overlay_count specifies the number of restart outputs
462  * which will be overlaid on top of the currently written
463  * step before advancing to the next step on the restart
464  * database.
465  *
466  * For example, if restarts are being output every 0.1
467  * seconds and the overlay count is specified as 2, then
468  * restart will write time 0.1 to step 1 of the database.
469  * It will then write 0.2 and 0.3 also to step 1. It will
470  * then increment the database step and write 0.4 to step 2;
471  * overlay 0.5 and 0.6 on step 2... At the end of the
472  * analysis, assuming it runs to completion, the database
473  * would have times 0.3, 0.6, 0.9, ... However, if there
474  * were a problem during the analysis, the last step on the
475  * database would contain an intermediate step.
476  *
477  * The cycle_count specifies the number of restart steps
478  * which will be written to the restart database before
479  * previously written steps are overwritten.
480  *
481  * For example, if the cycle count is 5 and restart is written every 0.1
482  * seconds, the restart system will write data at times 0.1, 0.2, 0.3,
483  * 0.4, 0.5 to the database.
484  *
485  * It will then overwrite the first step with data from time 0.6, the
486  * second with time 0.7. At time 0.8, the database would contain data at
487  * times 0.6, 0.7, 0.8, 0.4, 0.5. Note that time will not necessarily be
488  * monotonically increasing on a database that specifies the cycle
489  * count.
490  *
491  * The cycle count and overlay count can both be used at the same time
492  * also. The basic formula is:
493  *
494  * db_step = (((output_step - 1) / overlay) % cycle) + 1
495  *
496  * where "output_step" is the step that this would have been on the
497  * database in a normal write (1,2,3,....) and "db_step" is the step
498  * number that this will be written to.
499  *
500  * If you only want the last step available on the database,
501  * use set_cycle_count(1)
502  */
503  void set_cycle_count(int count) const { cycleCount = count; }
504  int get_cycle_count() const { return cycleCount; }
505  void set_overlay_count(int count) const { overlayCount = count; }
506  int get_overlay_count() const { return overlayCount; }
507  void set_file_per_state(bool yes_no) const { filePerState = yes_no; }
508  bool get_file_per_state() const { return filePerState; }
