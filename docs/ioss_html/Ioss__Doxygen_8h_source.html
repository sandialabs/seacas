28 ## Auto-Decomposition-Related Properties
29 
30  Property | Value | Description
31 -----------------|:------:|-----------------------------------------------------------
32 MODEL_DECOMPOSITION_METHOD | {method} | Decompose a DB with type `MODEL` using `method`
33 RESTART_DECOMPOSITION_METHOD | {method} | Decompose a DB with type `RESTART_IN` using `method`
34 DECOMPOSITION_METHOD | {method} | Decompose all input DB using `method`
35 PARALLEL_CONSISTENCY | [on]/off | On if the client will call Ioss functions consistently on all processors. If off, then the auto-decomp and auto-join cannot be used.
36 RETAIN_FREE_NODES | [on]/off | In auto-decomp, will nodes not connected to any elements be retained.
37 LOAD_BALANCE_THRESHOLD | {real} [1.4] | CGNS-Structured only -- Load imbalance permitted Load on Proc / Avg Load
38 
39 ### Valid values for Decomposition Method
40 
41 Method | Description
42 :---------:|-------------------
43 rcb | recursive coordinate bisection
44 rib | recursive inertial bisection
45 hsfc | hilbert space-filling curve
46 metis_sfc | metis space-filling-curve
47 kway | metis kway graph-based
48 kway_geom | metis kway graph-based method with geometry speedup
49 linear | elements in order first n/p to proc 0, next to proc 1.
50 cyclic | elements handed out to id % proc_count
51 random | elements assigned randomly to processors in a way that preserves balance (do not use for a real run)
52 external | Files are decomposed externally into a file-per-processor in a parallel run.

114 ## Setting properties via an environment variable
115 
116 Although the properties are usually accessed internally in the
117 application calling the IOSS library, it is possible to set the
118 properties externally prior to running the application via the setting
119 of the environment variable `IOSS_PROPERTIES`. The value of the
120 variable is one or more colon-separated property/property-value pairs.
121 For example, to set the `DECOMPOSITION_METHOD` and the `FILE_TYPE`
122 externally, the following would be used:
123 ```
124  export IOSS_PROPERTIES="DECOMPOSITION_METHOD=rib:FILE_TYPE=netcdf4"
125 ```
126 If the environment variable is set correctly, there should be an
127 informational message output during running of the application similar
128 to:
129 ```
130  IOSS: Adding property 'DECOMPOSITION_METHOD' with value 'rib'
131  IOSS: Adding property 'FILE_TYPE' with value 'netcdf4'
132 ```
