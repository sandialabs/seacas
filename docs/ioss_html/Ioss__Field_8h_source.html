47  /* \brief Categorizes the type of information held in the field.
48  */
49  enum RoleType {
50  INTERNAL,
51  MESH, /**< A field which is used to define the basic geometry
52  or topology of the model and is not normally transient
53  in nature. Examples would be element connectivity or
54  nodal coordinates. */
55  ATTRIBUTE, /**< A field which is used to define an attribute on an
56  EntityBlock derived class. Examples would be thickness
57  of the elements in a shell element block or the radius
58  of particles in a particle element block. */
59  COMMUNICATION,
60  MESH_REDUCTION, /**< A field which summarizes some non-transient data
61  about an entity (\sa REDUCTION). This could be an
62  offset applied to an element block, or the units
63  system of a model or the name of the solid model
64  which this entity is modelling... */
65  INFORMATION = MESH_REDUCTION,
66  REDUCTION, /**< A field which typically summarizes some transient data
67  about an entity. The size of this field is typically not
68  proportional to the number of entities in a GroupingEntity.
69  An example would be average displacement over a group of
70  nodes or the kinetic energy of a model. This data is also
71  transient. */
72  TRANSIENT /**< A field which is typically calculated at multiple steps
73  or times in an analysis. These are typically "results"
74  data. Examples would be nodal displacement or element
75  stress. */
76  };
